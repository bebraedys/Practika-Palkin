# Инструкция по установке и запуску приложения Palkin

## Требования
- PostgreSQL 12+
- .NET 8.0 SDK
- Visual Studio 2022 (опционально)

## Шаг 1: Настройка PostgreSQL

### 1.1 Создание роли и базы данных

Выполните следующие команды в pgAdmin или через psql:

```sql
-- Создание роли app с паролем
CREATE ROLE app WITH LOGIN PASSWORD '123456789';

-- Создание базы данных
CREATE DATABASE "Palkin" OWNER app;
```

### 1.2 Выполнение скрипта создания таблиц

1. Подключитесь к базе данных `Palkin`
2. Выполните скрипт из файла `palkinlib/Scripts/CreateDatabase.sql`

Или через командную строку:
```bash
psql -U app -d Palkin -f "C:\Users\haermaeus_mora\source\repos\palkin\palkinlib\Scripts\CreateDatabase.sql"
```

## Шаг 2: Настройка приложения

### 2.1 Строка подключения

При необходимости измените строку подключения в файле `palkinprog/App.xaml.cs`:

```csharp
var connectionString = "Host=localhost;Port=5432;Database=Palkin;Username=app;Password=123456789";
```

## Шаг 3: Запуск приложения

### 3.1 Из Visual Studio 2022
1. Откройте решение `palkin.sln`
2. Нажмите F5 или кнопку "Start"

### 3.2 Из командной строки
```bash
cd C:\Users\haermaeus_mora\source\repos\palkin
dotnet run --project palkinprog
```

## Шаг 4: Запуск тестов

```bash
dotnet test palkinlibtests
```

## Функционал приложения

### Главное окно
- Просмотр списка всех партнеров
- Сортировка по колонкам
- Поиск по таблице

### Контекстное меню (правый клик)
- **Редактировать** - открытие формы редактирования партнера
- **История продаж** - просмотр истории продаж партнера
- **Удалить** - удаление партнера (с подтверждением)

### Добавление партнера
1. Нажмите кнопку "+ Добавить партнера"
2. Заполните обязательные поля (наименование, тип партнера)
3. Нажмите "Сохранить"

### Редактирование партнера
1. Дважды кликните на партнере в таблице или выберите "Редактировать" в контекстном меню
2. Измените данные
3. Нажмите "Сохранить"

### Просмотр истории продаж
1. Выберите партнера
2. Нажмите "История продаж" в контекстном меню
3. Просмотрите информацию о продажах, общей сумме и скидке

## Система скидок

Скидка рассчитывается автоматически на основе общей суммы продаж партнера:

| Общая сумма продаж (₽) | Скидка | Статус |
|------------------------|--------|--------|
| до 10 000 | 0% | Стандарт |
| 10 000 - 49 999 | 5% | Серебряный партнер |
| 50 000 - 299 999 | 10% | Золотой партнер |
| от 300 000 | 15% | Премиум партнер |

## Возможные ошибки и решения

### Ошибка подключения к базе данных
- Проверьте, запущен ли PostgreSQL
- Проверьте правильность строки подключения
- Убедитесь, что роль `app` создана с правильным паролем

### Ошибка при загрузке данных
- Убедитесь, что скрипт CreateDatabase.sql выполнен
- Проверьте наличие данных в таблице partner_types
